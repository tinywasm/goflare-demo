## Setup: install `ormc`

```bash
go install github.com/tinywasm/orm/cmd/ormc@latest
```

Run from the project root after creating `web/model.go`:

```bash
ormc
```

This scans for `model.go` / `models.go` and generates `model_orm.go` next to each file.
`ormc` also runs `go get` and `go mod tidy` automatically.

## `modules/contact/model.go`

This is the **input** file ‚Äî written by hand. `ormc` scans `modules/` and generates
`modules/contact/model_orm.go` next to it.

No build tag ‚Äî `model.go` is build-agnostic. It is used by frontend WASM, edge WASM,
and local dev server alike. `ormc` places no tag restrictions on the input file.

The directive `// ormc:formonly` tells `ormc` to generate Form layer only (no DB).
`NotNull` and widgets are controlled via `input:` tags.

```go
//go:build wasm

package contact

// ormc:formonly
type ContactForm struct {
    Nombre  string `input:"required,min=2"`
    Email   string `input:"email,required"`
    Mensaje string `input:"textarea,required,min=10"`
}
```

After running `ormc`, the generated `modules/contact/model_orm.go` will contain:
- `Schema() []fmt.Field` ‚Äî field descriptors with widgets and validation rules
- `Pointers() []any` ‚Äî field pointers for form binding
- `Validate(action byte) error` ‚Äî validation using `fmt.ValidateFields`

> **Note:** `FormName()` does not exist in tinywasm/orm. `ModelName()` is generated only
> for DB structs (not `formonly`). Form identification is handled by the form name passed
> to `form.New("app", data)` in `client.go`.

## `web/client.go`

`WORKER_URL` is read from `window.WORKER_URL` set in `index.html` (see below).
`dom.Render` is used instead of the non-existent `dom.SetInnerHTML`.

```go
//go:build wasm

package main

import (
    "syscall/js"

    "github.com/tinywasm/dom"
    "github.com/tinywasm/fetch"
    "github.com/tinywasm/fmt"
    "github.com/tinywasm/form"
    "github.com/tinywasm/json"
)

func main() {
    // Worker URL set in index.html as window.WORKER_URL.
    // Fallback to "/contact" for local development with proxy.
    workerURL := js.Global().Get("WORKER_URL").String()
    if workerURL == "" {
        workerURL = "/contact"
    }

    data := &ContactForm{}

    f, err := form.New("app", data)
    if err != nil {
        fmt.Println("form error:", err)
        return
    }

    f.OnSubmit(func(fielder fmt.Fielder) error {
        var body []byte
        if err := json.Encode(data, &body); err != nil {
            return err
        }

        fetch.Post(workerURL).
            ContentTypeJSON().
            Body(body).
            Send(func(resp *fetch.Response, err error) {
                if err != nil {
                    dom.Render("result", dom.P("Error: "+err.Error()))
                    return
                }
                dom.Render("result", dom.P("¬°Mensaje enviado!"))
            })

        return nil
    })

    if err := dom.Render("app", f); err != nil {
        fmt.Println("render error:", err)
        return
    }

    select {}
}
```

## `edge/main.go`

- Directory must be `edge/` ‚Äî goflare auto-detects `edge/main.go` and sets `Entry = "edge"` automatically. No `ENTRY=` needed in `.env`.
- Imports `contact.ContactForm` from `modules/contact/` ‚Äî no duplicate struct definitions.
- Validation via `data.Validate(0)` ‚Äî uses the `ormc`-generated method, no manual field checks.
- `Content-Type` is set per-branch, not globally (CORS preflight must not have a body).

```go
//go:build wasm

package main

import (
    "github.com/tinywasm/fmt"
    "github.com/tinywasm/goflare/workers"
    "github.com/tinywasm/json"

    "github.com/tinywasm/goflare-demo/modules/contact"
)

type jsonMsg struct {
    Key   string
    Value string
}

func (m *jsonMsg) Schema() []fmt.Field {
    return []fmt.Field{{Name: m.Key, Type: fmt.FieldText}}
}

func (m *jsonMsg) Pointers() []any { return []any{&m.Value} }

func main() {
    workers.Handle(handler)
}

func handler(w *workers.Response, r *workers.Request) {
    w.Header()["Access-Control-Allow-Origin"] = "*"

    if r.Method == "OPTIONS" {
        // CORS preflight ‚Äî no body, no Content-Type
        w.Header()["Access-Control-Allow-Methods"] = "POST, OPTIONS"
        w.Header()["Access-Control-Allow-Headers"] = "Content-Type"
        w.WriteHeader(204)
        return
    }

    // All responses with body are JSON
    w.Header()["Content-Type"] = "application/json"

    writeJSON := func(status int, key, value string) {
        w.WriteHeader(status)
        json.Encode(&jsonMsg{Key: key, Value: value}, w)
    }

    if r.Method != "POST" {
        writeJSON(405, "error", "method not allowed")
        return
    }

    var data contact.ContactForm
    if err := json.Decode(r.Body(), &data); err != nil {
        writeJSON(400, "error", "invalid json")
        return
    }

    if err := data.Validate(0); err != nil {
        writeJSON(422, "error", err.Error())
        return
    }

    // TODO: send email notification using tinywasm/fetch to Resend/Mailgun
    // fetch.Post("https://api.resend.com/emails").
    //     Header("Authorization", "Bearer "+env.Get("RESEND_API_KEY")).
    //     ContentTypeJSON().Body(emailPayload).Send(...)

    writeJSON(200, "message", "¬°Mensaje recibido! Te contactamos pronto.")
}
```
