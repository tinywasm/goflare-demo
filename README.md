# GoFlare Demo: Contact Form
<img src="docs/img/badges.svg">

This is a demonstration project using the **GoFlare** stack to build a high-performance web application with a Go WASM frontend and a Cloudflare Workers backend.

## Features

- **Frontend WASM**: Built with `tinywasm/dom` and `tinywasm/form`.
- **Backend Edge**: Cloudflare Workers powered by `tinywasm/goflare/workers`.
- **Validation**: Shared logic and schemas using `tinywasm/fmt` and `tinywasm/orm`.
- **Styling**: Modern, responsive design with Vanilla CSS.

## Project Structure

- `modules/contact/`: Shared models and form schemas.
- `web/`: Frontend source (WASM).
  - `client.go`: Main entry point for the WASM application.
  - `public/`: Static assets (HTML, CSS).
- `worker/`: Backend source (Cloudflare Worker).
  - `main.go`: Edge function handler.
- `.build/`: Build artifacts generated by `goflare build`.

## Local Development

1. **Install Dependencies**:
   ```bash
   go mod tidy
   ```

2. **Generate ORM Code**:
   ```bash
   ormc
   ```

3. **Build Artifacts**:
   ```bash
   goflare build
   ```

4. **Run Local Server**:
   ```bash
   go run web/server.go
   ```
   Open `http://localhost:8080` in your browser.

## Deployment

1. **Configure Environment**:
   Update `.env` with your `CLOUDFLARE_ACCOUNT_ID`.

2. **Deploy**:
   ```bash
   goflare deploy
   ```

3. **Finalize**:
   Update `window.WORKER_URL` in `web/public/index.html` with your deployed worker URL.
